# Licensing

HydraKVM is dual-licensed.

## Open source license: AGPLv3

The default license is the **GNU Affero General Public License version 3
(AGPLv3)**, the full text of which is in [`LICENSE`](LICENSE).

This is what applies to:

- Personal, hobbyist, and homelab use
- Internal use within an organization, provided source modifications are made
  available per AGPLv3 to network-interacting users
- Forking, modifying, and redistributing under AGPLv3 (with the attribution
  and source-disclosure obligations the license specifies)

Under AGPLv3, if you run a modified HydraKVM as part of a network-accessible
service, you must offer the source of your modifications to users of that
service. This is the key way AGPLv3 differs from the regular GPL.

## What counts as "commercial"?

Selling hardware that includes HydraKVM, offering a paid service built on
HydraKVM, or building HydraKVM into a closed-source product is commercial
use that requires a commercial license (or full AGPLv3 compliance).

Using it personally, in a homelab, or within an organization for internal
IT operations is not commercial use in this sense and is fully covered by
AGPLv3.

If you're unsure which side of the line your use case falls on, ask.
